## Vietnamese Automatic Speech Recognition System
**Python · C++ · HTML5/CSS3/JavaScript · Docker** | 2020

Adapted an ASR system to accurately recognise Vietnamese speech. The project covered the full pipeline: data collection (in collaboration with the blind association), model training, inference API development, and a browser-based demo application. A custom Vietnamese dictionary was also constructed from processed corpus text.
